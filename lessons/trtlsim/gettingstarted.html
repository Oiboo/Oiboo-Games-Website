Initial Set-up

First downlaod the project off github

If you havn't already, download the Unreal engine .

Download the launcher from the website.

From the launcher, download Engine version 4.19.2 by clicking the plus icon next to engine versions

Once The Unreal Engine has been installed you can open the project.

Click the Turtle Simulator.Uproject file in the file browser

Unreal engine will build the necessary files before opening the project. Click yes to build the .dll

It will also mention that the Loading plugin was built with an older version. Click Yes
