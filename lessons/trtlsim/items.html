BP Folder

This folder contains the rest of the interactable items in the game.

The chests, Coral, Oxygen vents, Race rings, can all be found in here

Each folder should contain the Blueprint, 3D model, UV map and Texture

Blueprint

Drag the Blueprint into the scene to use the object

Some of the objects have default variables that must be set

For example

After dragging a chest into the scene you want to put something in it

check in the Default tab and you will see a drop down box

From there we can select what we want to spawn inside the chest

note. not all items will spawn, i havn't tried everything yet
